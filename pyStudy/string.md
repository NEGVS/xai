Python 中的字符串拼接有多种方式，不同方式适用于不同场景。我会从最基础到最推荐的方式逐一讲解，帮你理解每种方法的用法和优缺点。

### 1. 最基础：使用 `+` 运算符
这是最直观的方式，直接用 `+` 把多个字符串连接起来。

```python
# 基础用法
str1 = "Hello"
str2 = "World"
result = str1 + " " + str2
print(result)  # 输出：Hello World

# 注意：+ 只能拼接字符串，不能直接拼接其他类型（如数字）
# 错误示例：print("年龄：" + 18)  # 会报错
# 正确做法：先把非字符串类型转为字符串
age = 18
print("年龄：" + str(age))  # 输出：年龄：18
```

### 2. 高效简洁：使用 `,` 逗号（仅限 print 输出）
在 `print` 函数中，用逗号分隔多个值，会自动用空格拼接并输出（但本质上不是真正的字符串拼接，只是输出时的便捷写法）。

```python
name = "小明"
score = 95
print("姓名：", name, "成绩：", score)  # 输出：姓名： 小明 成绩： 95
```

### 3. 批量拼接：使用 `join()` 方法（推荐）
当需要拼接**多个字符串（如列表/元组中的字符串）** 时，`join()` 是效率最高的方式（比多次用 `+` 快很多）。

语法：`分隔符.join(可迭代对象)`（可迭代对象如列表、元组，且元素必须都是字符串）。

```python
# 拼接列表中的字符串
words = ["Python", "字符串", "拼接"]
# 用空字符串拼接（无分隔符）
result1 = "".join(words)
print(result1)  # 输出：Python字符串拼接

# 用指定分隔符拼接（如空格、逗号）
result2 = " - ".join(words)
print(result2)  # 输出：Python - 字符串 - 拼接

# 如果列表中有非字符串元素，需先转换
nums = [1, 2, 3]
result3 = "|".join(str(num) for num in nums)
print(result3)  # 输出：1|2|3
```

### 4. 灵活易用：格式化字符串（f-string / format）
不仅能拼接，还能灵活嵌入变量、表达式，是日常开发中**最推荐**的方式（Python 3.6+ 支持 f-string）。

#### 方式 4.1：f-string（简洁高效）
```python
name = "小红"
height = 1.65
# 直接在字符串前加 f，用 {} 嵌入变量/表达式
result = f"姓名：{name}，身高：{height} 米，身高厘米数：{height * 100}"
print(result)  # 输出：姓名：小红，身高：1.65 米，身高厘米数：165.0
```

#### 方式 4.2：`str.format()`（兼容低版本 Python）
```python
fruit = "苹果"
price = 5.99
# 用 {} 占位，后续用 format 传值
result = "{}的价格是{}元/斤".format(fruit, price)
print(result)  # 输出：苹果的价格是5.99元/斤
```

### 5. 老式方式：`%` 格式化（不推荐，仅了解）
这是 Python 早期的格式化方式，可读性不如 f-string，现在很少用。

```python
animal = "猫"
count = 2
result = "我家有%d只%s" % (count, animal)
print(result)  # 输出：我家有2只猫
```

---

### 总结
1. **少量简单拼接**：用 `+` 运算符（直观，但多次拼接效率低）。
2. **批量拼接多个字符串**：用 `join()` 方法（效率最高，适合列表/元组场景）。
3. **日常开发（灵活嵌入变量/表达式）**：优先用 **f-string**（简洁、高效、易读）。
4. **核心注意点**：`+` 只能拼接字符串，非字符串类型需先用 `str()` 转换。
